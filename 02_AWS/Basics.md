# How a Website Works
Client → Network → Server
- Client uses a network (browser + internet) to send request
- Client sends data packet (data)
- Server accepts the request
- Server processes it and sends response
- Client receives response and displays website

## What Server Composed of ?
- Compute: CPU  
- Memory: RAM  
- Storage: Database  
- Network: Router, Switch, DNS Server
- Also they will be requiring IP address

# Network
- Routers, cables and servers are connected with each other 
# Router
- A networking device that forward data packets between computer networks 
# Switch 
- Switch sends packet to correct server/client  

## Problems with Data Centers
- Power supply, cooling maintenance  
- Scaling is limited  
- Hardware failures  
- Monitoring overhead  
- Solution: Use Cloud
