# Linux Project 3: Log Automation Script

## 📌 Objective
The goal of this project is to automate log analysis using a shell script instead of manually running commands.

## What I Did
* Created a shell script (`analyze_logs.sh`)
* Automated log analysis using bash scripting
* Counted ERROR and WARNING logs
* Displayed filtered log output automatically

## Script Functionality
* Reads log file (`app.log`)
* Counts number of ERROR logs
* Counts number of WARNING logs
* Displays all ERROR logs
* Prints a structured report

## What I Learned
* Basics of shell scripting
* Using variables in bash
* Automating repetitive tasks
* Real-world DevOps log monitoring approach

## Project Structure
* analyze_logs.sh → automation script
* app.log → sample log file

## Conclusion
This project helped me understand how automation works in DevOps and how shell scripting can simplify repetitive tasks like log analysis.
